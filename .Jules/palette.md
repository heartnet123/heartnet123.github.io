## 2025-05-15 - [Keyboard-Accessible Tooltips]
**Learning:** Tooltips that only appear on hover are a common accessibility barrier for keyboard users. By using CSS classes like `group-focus-visible:opacity-100` in conjunction with `group-hover:opacity-100`, we can ensure that critical navigational context is available to all users.
**Action:** Always pair hover-based visibility triggers with focus-based triggers for any decorative or informative tooltips that accompany icon-only interactive elements.
